# 📋 Employee Management System (MERN Stack)

A full-stack web application designed to manage employee records securely. This project was developed as part of **Task 2** of my internship, focusing on CRUD operations and secure administrative access.

---

## 🚀 Key Features
* **Secure Authentication:** Restricted access via a custom login system (Admin: `shreyanka`).
* **Full CRUD Functionality:** Ability to **Create, Read, Update, and Delete** employee records in real-time.
* **Persistent Database:** Integrated with **MongoDB** for reliable data storage.
* **Modern UI:** A clean, responsive dashboard built with **React** and custom CSS.

## 🛠️ Tech Stack
* **Frontend:** React.js, Axios
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Security:** Password verification and protected state management.

---
